Installation#

This page provides the recommended installation flow for FGManip and commonly used training frameworks.

Prerequisites#

  • Python 3.9+ (3.10 recommended)

  • PyTorch compatible with your CUDA / system configuration

  • ManiSkill runtime dependencies (via mani_skill)

Install FGManip#

# create and activate an environment (example)
conda create -n fgmanip python=3.10 -y
conda activate fgmanip

# install pytorch for your platform first
pip install torch torchvision torchaudio

# install ManiSkill base runtime
pip install --upgrade mani_skill

# install FGManip
git clone https://github.com/YOUR_ORG/FGManip.git
cd FGManip
pip install -e .

Optional Training Frameworks#

We provide training and evaluation examples for both LeRobot and StarVLA. If you plan to run those pipelines, install the corresponding framework.

LeRobot (optional)

# recommended install
pip install -e ".[pi]"

# for specific versions, use the official repo install
pip install "lerobot[pi]@git+https://github.com/huggingface/lerobot.git"

StarVLA (optional)

conda create -n starVLA python=3.10 -y
conda activate starVLA
# then follow StarVLA official installation steps

For exact version compatibility and troubleshooting, refer to the official project docs:

  • LeRobot

  • StarVLA

Verify Installation#

Run a minimal sanity check from the FGManip repo root:

python -c "import mani_skill; print('ManiSkill OK')"
python -c "import core; print('FGManip core OK')"

Then continue to Quickstart for the end-to-end data workflow.

More Help#

For system-level setup issues (drivers, Vulkan, platform compatibility), please refer to the official ManiSkill installation documentation:

https://maniskill.readthedocs.io/en/latest/getting_started/installation.html
